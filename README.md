# Autonuke
Nuke goes off after x amount of time.
# Config Options
| Name | Type | Description | Default |
| --- | --- | --- | --- |
| `AutoNukeTime` | float | Toggles the plugin | 600 |

This plugin requires [Synapse](https://github.com/SynapseSL/Synapse)
